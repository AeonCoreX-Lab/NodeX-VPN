# @aeoncorex/nodex-vpn-darwin-arm64

Platform-specific binary package for **NodeX VPN** CLI.

**Platform:** `darwin / arm64`
**Rust target:** `aarch64-apple-darwin`

This package is installed automatically as an optional dependency of
[`@aeoncorex/nodex-vpn`](https://www.npmjs.com/package/@aeoncorex/nodex-vpn).

Do not install this package directly.
